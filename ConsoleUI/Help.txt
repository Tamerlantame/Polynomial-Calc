Допустимые команды:
help              - выводит помощь
create            - создаёт граф, требуется указать путь к текстовому файлу, содержащему матрицу смежности и имя графа
change            - делает активным указаныый граф
write             - выписывает граф
get diam          - выводит диаметр графа
add node          - добавляет вершину, требует ввести через пробел строку с номерами соседних и соседствующих вершин
add edge          - добавляет ребро, требует ввести номер вершины начала и номер вершины конца
save              - сохраняет граф в указанной папке
exit              - завершение работы
